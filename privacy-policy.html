Privacy Policy for Github Notifications Extension

Last updated: August 1, 2025

Information We Collect

The Github Notifications extension collects and stores the following information locally on your device:

  • GitHub Personal Access Token: Required to authenticate with the GitHub API and retrieve your pull request notifications
  • GitHub Repository Information: Names and details of repositories you configure for monitoring
  • Notification Preferences: Your settings for how often to check for notifications and display preferences

How We Use Your Information

Your information is used solely to:

  • Connect to the GitHub API using your personal access token
  • Retrieve and display your pull request notifications
  • Store your preferences and settings locally

Data Storage and Security

  • Local Storage Only: All data is stored locally on your device using Chrome's extension storage API
  • No External Servers: We do not transmit your data to any external servers or third parties
  • GitHub API Only: The extension only communicates with GitHub's official API (api.github.com)

Data Sharing

We do not share, sell, or transmit your personal information to any third parties. Your GitHub token and repository information remain on your device and are only used to communicate directly with GitHub's API.

Data Retention

Your data is stored locally until you:

  • Uninstall the extension
  • Clear the extension's data through Chrome settings
  • Reset the extension settings

Your Rights

You have full control over your data:

  • View your stored data through the extension's options page
  • Delete your data by uninstalling the extension or clearing storage
  • Modify your settings and preferences at any time

Third-Party Services

This extension interacts only with GitHub's API. Please refer to GitHub's Privacy Policy for information about how GitHub handles your data.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above and in updated versions of the extension.

Contact

If you have any questions about this privacy policy, please create an issue on our GitHub repository.

Compliance

This extension is designed to comply with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
